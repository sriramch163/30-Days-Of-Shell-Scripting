## Day 10 – Public and Private IP Finder

This script displays the machine's public IP and private IP addresses.

Why this is useful:
Helps DevOps engineers quickly identify network configuration of servers.

Usage:
./day-10-ip-check.sh

Example Output
==============================
   Network Information
==============================

Public IP Address:
34.93.210.45

Private IP Address:
10.128.0.15

IP check completed
