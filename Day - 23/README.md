## Day 23 – Git Merged Branch Cleanup

This script deletes local Git branches that have already been merged.

Why this is useful:
Keeps the repository clean by removing unnecessary branches.

Usage:
./day-23-git-clean-branches.sh

==============================
 Git Merged Branch Cleanup
==============================

Deleting branch: feature/login
Deleting branch: bugfix/api-error
