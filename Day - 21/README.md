## Day 21 – Kubernetes Pod Status Dashboard

This script summarizes pod statuses across all namespaces.

Why this is useful:
Provides a quick overview of cluster health and helps detect failing pods.

Usage:
./day-21-pod-status-dashboard.sh




Example Output
==============================
 Kubernetes Pod Status Dashboard
==============================

Running              15
Pending              2
CrashLoopBackOff     3
Failed               1

Status summary completed
