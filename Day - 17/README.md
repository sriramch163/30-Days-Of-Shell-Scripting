## Day 17 – Docker Image Builder with Timestamp

This script builds a Docker image and tags it using the current timestamp.

Why this is useful:
Ensures every Docker image build has a unique version tag.

Usage:
./day-17-docker-build.sh <image-name>

Example Output

Building Docker image...
Image Tag: myapp:20260316-1015

[Docker build output...]

Docker image built successfully: myapp:20260316-1015
