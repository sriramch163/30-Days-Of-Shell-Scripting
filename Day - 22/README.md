## Day 22 – Git Auto Pull Script

This script automatically pulls the latest changes for multiple Git repositories.

Why this is useful:
Saves time when working with multiple projects by updating them in one command.

Usage:
./day-22-git-auto-pull.sh


Example Output
Updating repo: /home/user/project1
Already up to date.
----------------------------------

Updating repo: /home/user/project2
Updating abc123..def456
----------------------------------

All repositories updated
